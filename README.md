# Image_Stegnography_System
A web-based application that securely hides secret messages in images using LSB and DCT techniques, enhanced with a machine learning steganalysis module to detect hidden data. Built using Python, Flask, and ML models.
This project implements a secure and intelligent image steganography system that allows users to hide confidential messages inside images using Least Significant Bit (LSB) and Discrete Cosine Transform (DCT) techniques.
